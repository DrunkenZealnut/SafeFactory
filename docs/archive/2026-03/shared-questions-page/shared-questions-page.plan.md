# Plan: 공유 질문 전용 페이지 (Shared Questions Page)

> **Feature**: shared-questions-page
> **Date**: 2026-03-13
> **Project**: SafeFactory

---

## Executive Summary

| Perspective | Description |
|-------------|-------------|
| **Problem** | 질문 공유·좋아요 기능(question-likes)이 이미 구현되어 있으나, 공유된 질문을 한눈에 탐색할 전용 페이지가 없어 도메인 페이지 하단의 "🔥 인기 질문 10개"와 워드클라우드 키워드 클릭으로만 접근 가능 |
| **Solution** | `/questions` 전용 페이지 신설 + `answer_full` 컬럼 추가로 AI 답변 전체를 저장하여, 질문 클릭 시 아코디언으로 답변을 즉시 표시 |
| **Function UX Effect** | (1) 전체 공유 질문 브라우징 (2) 질문 클릭→답변 아코디언 펼침 (3) 도메인·정렬 필터 (4) 좋아요 인라인 토글 (5) 내 질문 관리 (6) 네비게이션 탭 추가 |
| **Core Value** | 집단지성 허브 완성 — 공유 질문이 발견·탐색·재사용되는 선순환 구조 확립, 워드클라우드와 상호 보완 |

---

## 1. 문제 정의

### 1.1 현재 상태

| 항목 | 상태 |
|------|------|
| 백엔드 API | ✅ 5개 엔드포인트 완비 (`share`, `popular`, `like`, `delete`, `my`) |
| 워드클라우드 | ✅ `/wordcloud` 전용 페이지에서 키워드 시각화 |
| 도메인 인기 질문 | ✅ 도메인 페이지에 상위 10개 표시 |
| 전용 목록 페이지 | ❌ **없음** — 전체 질문 브라우징 불가 |
| 내 질문 관리 UI | ❌ **없음** — API만 존재 (`/api/v1/questions/my`) |

### 1.2 해결해야 할 문제

1. **발견성 부재**: 다른 도메인의 인기 질문을 보려면 각 도메인 페이지를 개별 방문해야 함
2. **탐색 불가**: 최신순·좋아요순 정렬, 키워드 검색 등 목록 탐색 기능이 없음
3. **내 질문 관리 불가**: 내가 공유한 질문을 확인·삭제할 UI가 없음
4. **워드클라우드 보완 필요**: 워드클라우드는 키워드 시각화에 특화, 질문 자체를 목록으로 보는 UX와 상호 보완

## 2. 기능 범위 (Scope)

### 2.1 In-Scope

| 기능 | 설명 |
|------|------|
| 전용 페이지 `/questions` | 공유 질문 전체 목록 페이지 |
| 도메인 필터 | 전체 / 반도체 / 현장실습 / 안전보건 / MSDS 탭 필터 |
| 정렬 옵션 | 인기순 (like_count DESC) / 최신순 (created_at DESC) |
| 페이지네이션 | 20개씩 로드, 더보기 버튼 또는 무한 스크롤 |
| 좋아요 토글 | 로그인 사용자 인라인 ♡/♥ 토글 (비로그인 시 로그인 유도) |
| 질문 클릭 | 해당 도메인 페이지로 이동 + 쿼리 파라미터로 AI 답변 자동 실행 |
| 내 질문 탭 | "내 질문" 탭에서 공유한 질문 확인·삭제 |
| 네비게이션 | base.html 상단 탭에 "❓ 질문" 추가 |

### 2.2 Out-of-Scope

- 질문 검색 (키워드 기반) — Phase 2
- 답변 미리보기 확장 (모달) — Phase 2
- 질문 신고 기능 — Phase 2
- 질문 태그/카테고리 — Phase 2

## 3. 기술 분석

### 3.1 기존 시스템 활용

| 기존 요소 | 활용 방식 |
|-----------|----------|
| `SharedQuestion` 모델 | 그대로 사용 — query, namespace, like_count, answer_preview, created_at |
| `QuestionLike` 모델 | 좋아요 상태 확인용 |
| `GET /api/v1/questions/popular` | 인기순 목록 API — **수정 필요**: 페이지네이션 파라미터 추가 |
| `POST /api/v1/questions/<id>/like` | 좋아요 토글 API 그대로 사용 |
| `GET /api/v1/questions/my` | 내 질문 목록 API 그대로 사용 |
| `DELETE /api/v1/questions/<id>` | 질문 삭제 API 그대로 사용 |
| `history.html` / `my_documents.html` | UI 패턴 참조 (필터 + 목록 + 페이지네이션) |

### 3.2 API 수정사항

| API | 수정 내용 |
|-----|----------|
| `GET /api/v1/questions/popular` | `page`, `per_page` 파라미터 추가, `sort` 파라미터 추가 (`likes` / `recent`) |

**기존 호환성**: `limit` 파라미터 방식은 유지하되 `page`/`per_page`가 있으면 paginated 응답 반환

### 3.3 신규 파일

| 파일 | 역할 |
|------|------|
| `templates/questions.html` | 공유 질문 전용 페이지 템플릿 |

### 3.4 수정 파일

| 파일 | 수정 내용 |
|------|----------|
| `web_app.py` | `/questions` 라우트 추가 |
| `templates/base.html` | 네비게이션 탭 "❓ 질문" 추가 |
| `api/v1/questions.py` | `popular` API에 페이지네이션·정렬 지원 추가 |

## 4. UI 설계

### 4.1 페이지 레이아웃

```
┌─────────────────────────────────────────────────┐
│  ❓ 공유 질문                                      │
│  다른 사용자들이 공유한 AI 질문을 탐색하세요           │
├─────────────────────────────────────────────────┤
│  [전체] [🔬반도체] [🏭현장실습] [🛡️안전보건] [🧪MSDS] │
│  정렬: [인기순 ▾]          [☁️ 워드클라우드 보기]    │
├─────────────────────────────────────────────────┤
│  ┌───────────────────────────────────────────┐  │
│  │ CVD 공정과 PVD 공정의 차이점은?            │  │
│  │ 🔬 반도체  ·  홍길동  ·  3일 전   ♥ 18   │  │
│  └───────────────────────────────────────────┘  │
│  ┌───────────────────────────────────────────┐  │
│  │ 중대재해처벌법 적용 대상은?                  │  │
│  │ 🛡️ 안전보건  ·  김철수  ·  1일 전   ♡ 22  │  │
│  └───────────────────────────────────────────┘  │
│  ...                                            │
│  [ ◀ 1 2 3 4 5 ▶ ]                              │
├─────────────────────────────────────────────────┤
│  [내 질문] 탭 (로그인 시)                          │
│  - 내가 공유한 질문 목록 + 삭제 버튼                 │
└─────────────────────────────────────────────────┘
```

### 4.2 질문 카드 구성

| 요소 | 내용 |
|------|------|
| 질문 텍스트 | `query` (클릭 시 도메인 페이지로 이동) |
| 도메인 뱃지 | namespace → 이모지+레이블 (🔬 반도체 등) |
| 작성자 | `author.name` |
| 시간 | timeAgo 형식 (3일 전, 1주 전 등) |
| 좋아요 | ♡/♥ + `like_count` (클릭으로 토글) |

### 4.3 기존 페이지 연동

| 연동 | 방식 |
|------|------|
| 워드클라우드 → 질문 페이지 | 워드클라우드 헤더에 "📋 질문 목록 보기" 링크 추가 |
| 질문 페이지 → 워드클라우드 | 질문 페이지 헤더에 "☁️ 워드클라우드 보기" 링크 추가 |
| 질문 클릭 → 도메인 페이지 | `/{domain}?q={encoded_query}` 이동 |

## 5. 사용자 스토리

### US-1: 전체 질문 탐색
> "모든 도메인의 공유 질문을 한곳에서 보고, 관심 있는 질문을 찾아 좋아요를 누르거나 AI에게 같은 질문을 하고 싶다."

**인수 조건**:
- `/questions` 접속 시 전체 공유 질문 인기순 표시
- 도메인 필터로 특정 도메인 질문만 필터링
- 인기순/최신순 정렬 전환

### US-2: 질문 클릭으로 AI 답변
> "인기 질문을 클릭하면 해당 도메인 페이지에서 바로 AI 답변을 받고 싶다."

**인수 조건**:
- 질문 클릭 시 `/{domain}?q={query}` 로 이동
- 도메인 페이지에서 쿼리 파라미터 감지 후 자동 AI 답변 실행

### US-3: 내 질문 관리
> "내가 공유한 질문 목록을 확인하고, 불필요한 질문은 삭제하고 싶다."

**인수 조건**:
- "내 질문" 탭 클릭 시 내가 공유한 질문 목록 (로그인 필수)
- 각 질문에 삭제 버튼 (확인 다이얼로그 후 삭제)
