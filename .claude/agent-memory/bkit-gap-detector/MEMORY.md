# Gap Detector Agent Memory

## Project: SafeFactory

### Architecture Patterns
- Python/Flask backend, no frontend framework (Jinja2 templates)
- Services layer under `services/`, CLI under `src/`
- Singleton pattern via `services/singletons.py` (double-checked locking with `threading.RLock`)
- Lazy imports used throughout service layer
- Error handling convention: try/except -> log warning -> return empty/safe default (never raise)

### Naming Conventions
- Files: snake_case.py
- Classes: PascalCase
- Constants: UPPER_SNAKE_CASE
- Log prefixes: `[ModuleName]` pattern (e.g., `[LawDRF]`, `[LegalRouter]`)
